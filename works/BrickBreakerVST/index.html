BrickBreakerVST

A Breakout-style game implemented as a VST3/AU audio plugin, controlled via MIDI Pitch Wheel. Features lock-free thread synchronization.

← Back to Works GitHub Download

Project Overview

This project is a VST3/AU plugin developed using the JUCE framework. It allows you to play a Breakout-style game directly within your DAW, controlling the paddle using the pitch bend wheel on a MIDI keyboard.

It implements a lock-free synchronization mechanism between the audio processing thread and the UI rendering thread, ensuring smooth gameplay without causing audio dropouts.
