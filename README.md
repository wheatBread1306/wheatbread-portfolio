# wheatBread1306 Portfolio Website

Welcome to the source code of my personal portfolio website! 

🌐 **Live Demo:** [https://wheatbread.dev/](https://wheatbread.dev/)

## Overview
This is a single-page, responsive, dark-themed portfolio website designed to showcase my projects, particularly audio plugins developed using the C++ JUCE framework. 

The site focuses on a minimalist aesthetic with smooth interactions, featuring a custom interactive wave animation in the background and a seamless language toggle between English and Japanese.

## Features
- **Dark Mode Design:** A sleek, modern user interface.
- **Interactive Canvas Animation:** A custom dynamic wave background implemented using HTML5 Canvas.
- **Bilingual Support:** Seamless toggling between English (Default) and Japanese using vanilla JavaScript.
- **Responsive Layout:** Optimized for both desktop and mobile viewing.
- **Zero Dependencies:** Built entirely with raw HTML, CSS, and Vanilla JavaScript. No heavy frameworks.

## Project Structure
- `index.html` - The main entry point containing the site structure.
- `css/style.css` - Custom styling and CSS variables for the dark theme.
- `js/script.js` - Logic for the language toggle and the wave canvas animation.
- `assets/` - Images, thumbnails, and the site favicon.
